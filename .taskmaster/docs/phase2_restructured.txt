# 第二阶段任务 - Tiny Room 网站开发

- 绘制完整 Sitemap／页面流程图
- 定义各页面数据模型（TypeScript interface / Prisma schema）

- 分页或无限滚动
- 标签过滤（可选）

## 3. UI 美化 (Priority: medium)

### 3.1 Dark 模式切换
- 设计暗色主题配色方案
- 实现 CSS 变量系统
- 添加主题切换按钮

### 3.2 加入艺术元素
- 设计并实现艺术字体
- 添加 GSAP/Framer Motion 动画效果

### 3.3 其他细节美化
- 优化按钮 hover 效果
- 美化滚动条
- 设计 404 页面
- 添加加载指示器

## 4. 数据更新功能 (Priority: high)

### 4.1 实现富文本上传显示

## 5. 性能优化 (Priority: medium)
- 图片懒加载实现
